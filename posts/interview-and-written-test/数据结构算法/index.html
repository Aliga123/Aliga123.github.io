哈希表#

  • 哈希表是一种基础的数据结构（键值对），很多其他的数据结构都是基于哈希表是实现的，例如字典dict、集合set等。

  • 哈希查找时间复杂度O(1)。

  • 做题一般可以使用dict来操作

Leetcode题目实操：

## 题目：两数之和

def fun(nums, target):
    num_dict = {}
    for index, number in enumerate(nums):
        diff = number - target
        
        if diff in num_dict:
            return [num_dict[diff], index]
        
        num_dict[diff] = index
    return []

栈#

  • 后进先出

Leetcode题目实操：

## 题目：检查括号时候匹配

def fun(s):
    stack = []
    
    map = {")":"(", "]": "[", "}": "{"}
    
    for c in s:
        if c in map:
            if not stack or stack[-1] != map[c]:
                return False
            stack.pop()
    	stack.append(c)
	
    return not stack

  • 做题使用class创建节点，使用迭代或递归将节点连起来
## 两两交换链表中的节点, 链表 + 递归

class node:
    def _init_(self, x):
        self.x = x
        self.next = None

def swap(head: ListNode) -> ListNode:
    if not head or not head.next:
        return head
    
    current = head.next
    head.next = swap(current.next)
    current.next = head
    
    return current

迭代法#

  • 函数循环。
def factorial_iterative(n):
    # 初始值为1
    result = 1
    # 从1到n（包括n）进行迭代
    for i in range(1, n + 1):
        result *= i  # 将当前的i乘到result上
    return result

# 测试函数
print(factorial_iterative(5))  # 输出应该是120

递归法#

  • 从外往内，反复在函数里面调用本函数。
## 题目：阶乘计算

def factorial(n):
    # 基本情况：如果n为0，则返回1
    if n == 0:
        return 1
    # 递归步骤：否则，返回n乘以(n-1)的阶乘
    else:
        return n * factorial(n-1)

# 测试函数
print(factorial(5))  # 输出应该是120

回溯法#

  • 在递归法的基础上，加上条件判断（不条件递归参数不一样），使得递归回退不止一条线。
## 题目：要求生成所有有效的括号组合

def generate_parentheses(n):
    def backtrack(s='', left=0, right=0):
        # 如果字符串长度等于2n，说明括号已经完整，添加到结果中
        if len(s) == 2 * n:
            result.append(s)
            return
        # 如果左括号的数量小于n，添加左括号
        if left < n:
            backtrack(s + '(', left + 1, right)
        # 如果右括号的数量小于左括号的数量，添加右括号
        if right < left:
            backtrack(s + ')', left, right + 1)

    result = []
    backtrack()
    return result

# 生成3对括号的所有有效组合
print(generate_parentheses(3))

双指针法#

  • 指针一前一后，列表数据指针为索引，链表数据指针为节点。
## 题目：移除列表元素（Remove Elements）
def removeElements(nums, val):
    slow = 0
    for fast in range(len(nums)):
        if nums[fast] != val:
            nums[slow] = nums[fast]
            slow += 1
    return slow

分治法#

  • 分而治之。 它将问题分解成若干个较小的子问题，然后递归地解决这些子问题，最后将子问题的解合并以形成原问题的解。
  • 分解–>解决–>合并。分解+递归
## 题目：最大子序列和

def max_crossing_sum(arr, low, mid, high):
    # 从中间往左边计算最大值
    left_sum = float('-inf')
    current_sum = 0
    for i in range(mid, low-1, -1):
        current_sum = current_sum + arr[i]
        if current_sum > left_sum:
            left_sum = current_sum
	# 从中间往右边计算最大值
    right_sum = float('-inf')
    current_sum = 0
    for i in range(mid+1, high+1):
        current_sum = current_sum + arr[i]
        if current_sum > right_sum:
            right_sum = current_sum
	# 合并
    return left_sum + right_sum

def max_subarray_sum(arr, low, high):
    if low == high:
        return arr[low]

    mid = (low + high) // 2
    left_sum = max_subarray_sum(arr, low, mid)
    right_sum = max_subarray_sum(arr, mid+1, high)
    cross_sum = max_crossing_sum(arr, low, mid, high)

    return max(left_sum, right_sum, cross_sum)

# 示例使用
arr = [-2, -3, 4, -1, -2, 1, 5, -3]
result = max_subarray_sum(arr, 0, len(arr) - 1)
print("Maximum contiguous sum is", result)

动态规划#

  • 初始状态–>状态转移方程–>终止值
## 题目：最大子序列和

class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        nums_len = len(nums)
        dp = [0] * nums_len
        dp[0] = nums[0]
        for i in range(1, nums_len):
            dp[i] = max(dp[i-1] + nums[i], nums[i])
            
        return max(dp)

深度优先搜索（DFS）#

  • 递归+循环实现
## 题目：子集。给一个整形数组，数组中的元素互不相同，返回该数组所有有可能的子集。

class Solution:
    def subsets(self, nums):
        def dfs(index, path):
            # 将当前路径添加到结果集中
            result.append(list(path))
            # 遍历数组中的剩余元素
            for i in range(index, len(nums)):
                # 选择当前元素
                path.append(nums[i])
                # 递归调用dfs，继续搜索
                dfs(i + 1, path)
                # 撤销选择，即回溯
                path.pop()

        result = []
        dfs(0, [])
        return result

# 示例使用
nums = [1, 2, 3]
solution = Solution()
print(solution.subsets(nums))

广度优先搜索（BFS）#

  • 队列+循环实现
## 题目：给你一个二叉树，请按层序遍历得到的节点值。

class Solution:
    def levelOrder(self, root: Optional[TreeNode]) -> List[List[int]]:
        if not root: return []
        queue = collections.deque()
        queue.append(root)
        result = []

        while queue:
            current = []
            for _ in range(len(queue)):
                node = queue.popleft()
                current.append(node.val)
                
                if node.left: queue.append(node.left)
                if node.right: queue.append(node.right)

            result.append(current)
        return result
